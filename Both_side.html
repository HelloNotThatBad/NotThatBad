Q :

回家的時候曾經和媽媽討論到九二共識等等，媽媽說她覺得早晚都要被統一，而且統一也沒啥不好的，你看看中國現在發展多好? 民主真的能當飯吃嗎?當面對媽媽這樣的問題時，我真的很生氣，但我不太知道該怎麼反駁他.....

A1 :

我覺得這個可以用舉例說故事的方式回覆媽媽。像是：馬麻～～妳知道郭台銘出來選總統誰最羨慕嫉妒嗎？是馬雲唷！馬雲辛辛苦苦創了阿里巴巴這個集團，但最後只能「被退休」，妳知道為什麼嗎？因為隨著阿里巴巴發展日益龐大，像是阿里巴巴旗下的支付寶深入中國人民生活各層面，因此在和中國銀行制度和國家法律產生的摩擦愈來愈多，讓中國政府擔心失去主導權，以致中國政府想換其集團領導階層體系，讓集團不會超出政府管控的範圍。反觀，身為臺灣人的郭台銘，同樣白手起家的商人，卻可以不用顧慮其公司會發展「過盛」，威脅到政府地位讓國家緊張，甚至還可以競逐總統參選人，公開對國家「喊話」還不會遭受生命威脅。同樣是商人，但哪一邊可以爽爽賺錢又不會擔心突然「被消失」，哪一邊的財產權才是真的賺在人民手中呢？（然後搭配親切的里長口氣）

Q :

【 六四事件——鎮壓六四天安門事件幫助中國更加穩定嗎？ 】
————————
因應六四事件三十年，你是否在和他人談論時，也有遇到以下的問題呢?
————————
有人說在那個年代，中國選擇武力鎮壓是正當的行為，他們為了社會的穩定不得不這樣做，不然會造成社會的動盪。
面對這樣的想法，讓我很疑惑明明武力鎮壓人民是一件多麼悲痛的事情，為何會有人覺得在那個世代是正確的決定呢?
我又要用什麼方式去告訴他，不管什麼年代用武力去對付手無寸鐵的人民都是錯誤的?
———————
想必在今天大家也接觸到許多六四事件的訊息，如果是你面對這樣的問題，你又會怎麼回答他呢?

A1 :

我覺得，會在意社會動盪，是希望自己和親朋好友能好好活著；而武力鎮壓，讓當時在場的人，不用等到社會動盪就已經失去生命了。

Q :

【 香港反送中遊行達103萬人 - 台灣該思考些什麼？ 】
https://www.nownews.com/news/20190609/3433043/?fbclid=IwAR18OISR0fwtQIHXLelBZJDhKT9xaioMfiqahuIBFjSVC_jR63PJh8H-X9I
————————
因應香港反送中遊行，你是否在和他人談論時，也有遇到以下的問題呢?
————————
有人說看看中國的經濟成長，若如同香港，我們採中國的一國兩制，也許能幫助提升台灣經濟
但這次的反送中遊行，《逃犯條例》中「移交中國受審」的修正，可能讓香港政府難以做出獨立、合法且符合人權的判斷，當年的「一國兩制」似乎也已變質。
看看香港的例子，再看看台灣，我該用什麼方式去告訴他，民主自由的價值正是台灣的珍貴之處，並且中國的「一國兩制」帶來的，可能不只有經濟起飛的美好，更是對基本人權的不夠保障
———————
想必在這兩天大家也接觸到許多香港反送中遊行的訊息，如果是你面對這樣的問題，你又會怎麼回答他呢?
1.司法、行政獨立（國家主權實質內涵之一）的重要性
2.此項修正案可能影響人民的基本權利-言論自由，你怎麼看

A1 :

也有人說：「這又不關台灣的事！」 但如果在香港的台灣人被中國「認定」有罪，甚至只是入境香港或在香港轉機，都有可能被移交中國受審 更何況，就算不關我們的事好了，我們本來就要培養國際觀啊，不然要當一輩子的井底之蛙嗎XD （而且身為地球村一份子，國際事務本來就牽一髮動全身，是要怎麼不關我們的事啦！）

A2 :

這次《送中條例》破壞香港的司法獨立性，也破壞當初簽約時承諾的一國兩制 沒有司法獨立，就沒有公正的審判，身為公權力之一司法制度若不公正，自然也就無法保障人權 有人說：「別做犯法的事就好了，擔心這個幹嘛？」 但是中國政府並不是民主制度下的政府，難道不會用「莫須有的罪名」或者「用一切方法讓你自願認罪」的方法嗎？（看看銅鑼灣書店案、李明哲，甚至戒嚴時期的台灣～）
